# ⚡ Mark's Game Shop

![Languages](https://img.shields.io/badge/Language-HTML5%2FJavaScript-orange)
![Frameworks](https://img.shields.io/badge/Frameworks-Firebase-yellow)
![Status](https://img.shields.io/badge/Status-Active-green)

Mark's Game Shop is a fast, secure, and user-friendly web application designed to facilitate affordable game top-ups for popular titles including Mobile Legends: Bang Bang (MLBB), Call of Duty, Roblox, and Bloodstrike. The platform provides a seamless ordering experience with real-time tracking, multi-server pricing support, and secure authentication.

## 🛠️ Core Competencies Demonstrated
*   **Full-Stack Web Development**: Built with a responsive, mobile-first design using HTML5, CSS3, and JavaScript.
*   **Authentication & Security**: Integrated Firebase Authentication (Google OAuth and Email/Password) to manage secure user sessions.
*   **Dynamic UI/UX**: Implemented complex, state-driven interfaces (tabs, modal overlays, toast notifications, skeleton loaders) for an engaging user experience.
*   **Complex Data Integration**: Developed dynamic pricing systems that adjust based on game servers and product categories.
*   **Payment & Order Workflow**: Designed a structured order management system with automated validation and tracking capabilities.

## 📂 Project Structure & Features
*   **`index.html`**: The main application file containing the core UI logic, state management, and integrated services.
*   **`login.html`**: A secondary authentication portal for user management.
*   **`/api`**: Backend logic handling API requests and server-side operations.
*   **`/next-app`**: A modular folder structure for future scalability and migration to modern frameworks.
*   **Assets**: Optimized local images and branding assets for GCash, GoTyme, Maya, and PayPal.

## 🚀 Getting Started
To run the project locally:
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Configure your Firebase settings in the `firebaseConfig` object located within the `<script>` tag in `index.html`.
